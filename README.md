## Set-up (2 Terminals):

### Step 1: Install Dependcies

- `npm run install:all`

### Step 2: Initalize Database

- `npm run init:db`

### Step 3: Run Backend Server (Terminal 1)

- `npm run dev:backend`

### Step 3: Run Frontend Server (Terminal 2)

- `npm run dev:frontend`
